## Adding a property

To add a new property:

1. Click **Add Property** from the portfolio dashboard or header
2. Fill in the property details form
3. Upload property photos (optional but recommended)
4. Click **Save**

The property is immediately added to your portfolio.

## Editing a property

Navigate to the property's detail page and click **Edit** to update:

- Property name and address
- Property type and unit count
- Photos and documents
- Notes

## Property overview page

Each property has a comprehensive overview page showing:

- **Key metrics** — Current occupancy, monthly revenue, outstanding work orders
- **Photos** — Property images you've uploaded
- **Tenant summary** — Active leases and rent status
- **Recent activity** — Latest work orders, documents, and compliance updates

## Deleting a property

<Callout type="warn">
  Deleting a property is permanent and removes all associated data including tenants, work orders, and documents. This action cannot be undone.
</Callout>

To delete a property, open the property page, click the menu (three dots) in the header, and select **Delete Property**. You'll be asked to confirm.
