## Creating a work order

To create a maintenance request:

1. Navigate to **Work Orders** in the sidebar, or open a property and go to the Work Orders tab
2. Click **New Work Order**
3. Select the property (if creating from the global view)
4. Describe the issue
5. Set a priority level: Low, Medium, High, or Urgent
6. Optionally assign a vendor
7. Click **Submit**

## Work order statuses

Work orders move through the following statuses:

- **Open** — Newly created, not yet started
- **In Progress** — A vendor has been assigned and work has begun
- **On Hold** — Work paused pending parts, approval, or scheduling
- **Completed** — Work finished and verified
- **Cancelled** — Work order voided

## Tracking progress

Each work order has a detail page where you can:

- Add updates and notes
- Upload photos of the issue or repair
- Track vendor costs
- Update the status
- Mark as completed

## Assigning vendors

When creating or editing a work order, you can assign a vendor from your directory. If the vendor isn't in your directory yet, you can add them directly from the work order form.
